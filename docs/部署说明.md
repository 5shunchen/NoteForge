# NoteForge 部署说明

## 📁 优化后的目录结构

```
noteforge/
├── src/                        # 源代码目录
│   ├── noteforge.html         # 主程序 HTML
│   ├── main.js                # Electron 主进程
│   └── package.json           # 应用配置
├── build/                      # 打包输出目录（生成后）
│   ├── NoteForge-Setup-4.1.0.exe  # 安装程序
│   └── win-unpacked/          # 未打包版本
├── docs/                       # 文档目录
│   ├── README.md              # 主文档
│   ├── 打包说明.md            # 打包指南
│   └── 使用说明.md            # 使用指南
├── package.json                # 项目配置（根目录）
├── 打包 exe.bat                # 一键打包脚本
├── 启动服务器.bat              # 本地服务器启动
├── LICENSE                     # MIT 许可证
└── .gitignore                 # Git 忽略文件
```

---

## 🚀 打包方法

### 一键打包（推荐）

1. 双击运行 `打包 exe.bat`
2. 等待自动完成（首次需下载约 200MB）
3. 安装包在 `build/NoteForge-Setup-4.1.0.exe`

### 手动打包

```bash
# 安装依赖
npm install

# 打包成安装程序
npm run build

# 打包成便携版
npm run build:portable
```

---

## 📦 安装包功能

### 自定义安装目录
- 安装过程中可选择任意安装路径
- 默认：`C:\Program Files\NoteForge`

### 注册到服务列表
- 自动添加到 Windows"应用和功能"
- 可通过控制面板完全卸载
- 卸载路径：`控制面板 > 程序 > NoteForge`

### 快捷方式
- ✅ 桌面快捷方式
- ✅ 开始菜单快捷方式
- ✅ 卸载程序快捷方式

## ⚠️ 注意事项

1. **首次打包**：需要下载 Electron 运行时（约 150MB）
2. **网络要求**：打包过程需要访问 npm 仓库
3. **系统要求**：Windows 10/11 x64

## 🐛 故障排除

### 打包失败
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules
rmdir /s /q node_modules

# 重新安装
npm install
```

### 安装失败
- 以管理员身份运行安装包
- 检查杀毒软件是否拦截
- 确保有足够的磁盘空间
