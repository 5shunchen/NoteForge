# NoteForge - 快速开始指南

## ✅ 已完成打包！

你的 NoteForge 已经打包成 Windows 应用程序了。

---

## 🚀 三种启动方式

### 1. 直接运行 EXE（最快）
```
双击：dist/NoteForge.exe
```

### 2. 使用启动脚本
```
双击：NoteForge.bat
```

### 3. 浏览器模式
```
双击：noteforge.html
```

---

## 📦 如何分享给他人

### 方式 A：发送整个 dist 文件夹
1. 将 `dist` 文件夹压缩成 ZIP
2. 发送给他人
3. 对方解压后运行 `NoteForge.exe`

### 方式 B：重新打包
1. 将项目文件夹压缩
2. 让对方运行 `打包 exe.bat`

---

## ⚙️ 首次运行提示

首次运行可能会提示安装 **WebView2 组件**：
- Windows 10/11 通常已预装
- 如无提示，可忽略
- 如需要，下载地址：https://developer.microsoft.com/en-us/microsoft-edge/webview2/
