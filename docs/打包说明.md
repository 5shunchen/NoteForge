# NoteForge 打包成 EXE 指南

## 📦 方法一：使用打包脚本（推荐）

### 前提条件
- 已安装 Node.js (https://nodejs.org/)

### 步骤
1. 双击运行 `打包 exe.bat`
2. 等待下载和打包完成（约 100MB）
3. 生成的 EXE 文件在 `dist` 文件夹中

### 结果
- **可执行文件**: `dist/NoteForge.exe`
- **便携版本**: 可将整个 `dist` 文件夹复制到其他电脑

---

## 📦 方法二：使用 Electron（需要网络）

### 步骤
1. 打开命令行，进入项目目录
2. 运行以下命令：

```bash
# 安装依赖
npm install

# 打包成 EXE
npm run build
```

### 结果
- **安装包**: `dist/NoteForge Setup.exe`
- **便携版**: `npm run build:portable`

---

## 📦 方法三：手动打包（最可靠）

### 使用 NW.js

1. 下载 NW.js
   - 访问：https://dl.nwjs.io/
   - 下载最新版本 `nwjs-vX.XX.X-win-x64.zip`

2. 解压到任意位置

3. 将 `noteforge.html` 放入解压目录

4. 将 `nw.exe` 重命名为 `NoteForge.exe`

5. 运行 `NoteForge.exe` 即可

---

## 📦 方法四：使用 Web2Exe 工具（图形界面）

1. 下载 Web2Exe
   - https://github.com/jaya98/Web2Exe/releases

2. 打开 Web2Exe

3. 配置：
   - **Source Files**: 选择 `noteforge.html`
   - **App Name**: NoteForge
   - **Version**: 4.1.0

4. 点击 "Build"

---

## 🔧 打包后优化

### 添加图标
1. 准备一个 `.ico` 格式图标文件
2. 命名为 `icon.ico`
3. 放入项目目录
4. 重新打包

### 减小体积
- 移除不必要的字体
- 压缩 HTML 文件
- 使用 CDN 加载 marked.js（需要网络）

## ⚠️ 注意事项

1. **首次运行**: 可能需要安装 WebView2 组件
   - Windows 10/11 通常已预装
   - 下载地址：https://developer.microsoft.com/en-us/microsoft-edge/webview2/

2. **数据保存**: 打包版数据仍保存在浏览器 localStorage
   - 不同电脑的数据不互通
   - 使用导入导出功能备份数据

3. **文件大小**:
   - NW.js 打包约 150MB
   - Electron 打包约 200MB
